### <u> 1. Customer Churn Prediction & Retention Modeling: 
* Development of an end-to-end predictive pipeline for customer churn detection
* Data cleaning, transformation, and validation using Pandas and Scikit-Learn
* Feature engineering focused on customer behavior and retention variables
* Application of Logistic Regression for churn prediction
* Evaluation of classification metrics (Accuracy, Precision, Recall, and F1-Score)
* Trade-off analysis between precision and recall for model optimization
* Interpretation of variables with the greatest impact on customer churn
* Analytical storytelling focused on retention and loyalty strategies <br>
* 👉 View Proyect (https://github.com/DiegoFCd/Customer-Churn-Prediction-Retention-Modeling-) <br>

### <u> 2. Retail High-Value Customer Prediction:
* Development of a predictive model for identifying high-value future customers
* Construction of a machine learning pipeline using Pandas and Scikit-Learn
* Application of RFM and feature engineering techniques to transactional behavior
* Implementation of temporal validation to prevent data leakage
* Strategic transformation of a CLV regression problem into a high-value customer classification model
* Handling class imbalances through class weighting and threshold tuning
* Evaluation of classification metrics and optimization: recall vs. precision
* Interpretation of behavioral patterns associated with high-value customers <br>
* 👉 View Proyect (https://github.com/DiegoFCd/Retail-High-Value-Customer-Prediction/tree/main) <br>
